## Quick Start

1. Clone and setup:
   ```bash
   git clone https://github.com/ramacharya06/credit-risk-predictor.git
   cd credit-risk-predictor
   pip install -r requirements.txt
   ```

2. Run the web app:
   ```bash
   streamlit run app.py
   ```
   This will open the interactive prediction interface in your browser.

## Features

- Interactive web interface with Streamlit
- Real-time credit risk predictions
- Data visualization dashboard
- Pre-trained Random Forest model

## Model Training

To retrain the model, run the notebooks in order:
1. `notebook/preprocessing_dataset.ipynb`
2. `notebook/model_training.ipynb`
